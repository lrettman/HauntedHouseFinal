Overview
This project is a modified version of an existing 3D stealth haunted house game, adapted and enhanced as part of my final project. The base game provided the core mechanics and environment, and my modifications focused on improving gameplay feedback, player experience, and visual presentation.
Features
Original Game
3D stealth-based haunted house environment


Player navigates through rooms while avoiding enemies


Core gameplay mechanics preserved


My Modifications
Post-Processing Enhancements


Added visual effects such as lighting tweaks, color grading, and bloom to improve atmosphere and immersion.


Timer System (Minor Mod)


Tracks how long the player takes to complete the game.


Timer displayed on screen for performance feedback.


Speed Boost (Major Mod)


Player can temporarily increase movement speed.


Visual icon appears above the player when speed boost is active.


Short duration with cooldown to maintain game balance.


Audio Enhancements


Added footstep sounds to improve player feedback while walking.


Audio cues synchronize with player movement for immersion.


How to Play
Run the game in Unity (or build the executable).


Navigate through the haunted house while avoiding enemies.


Watch the timer in the UI to track your completion time.


Use the speed boost when needed to escape or traverse faster (icon indicates active boost).


Enjoy the enhanced visuals and audio for a more immersive experience.


Controls
Movement: WASD / Arrow keys


Speed Boost: Left Shift (temporary)


Camera / Look: Mouse


Notes
This project demonstrates my ability to take an existing game, apply aesthetic improvements, and implement functional gameplay mods.


The speed boost and timer showcase both minor and major game modifications, adding new mechanics and player feedback.
